# Vitepress

VitePress is a Vite-powered static site generator.

[Read the docs](https://vitepress.dev/)

## Usage

> [!TIP]
>
> You need to have [pnpm](https://pnpm.io/) installed globally.

Install dependencies:

```bash
pnpm i
```

Preview:

```bash
pnpm run docs:dev
```

Build:

```bash
pnpm run docs:build
```
